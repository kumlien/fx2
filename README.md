# fx2


Formula used to calculate max units to buy
Margin Available * (margin ratio) / ({BASE}/{HOME Currency} Exchange Rate)
For example, suppose:
Home Currency: USD
Currency Pair: GBP/CHF
Margin Available: 100
Margin Ratio : 20:1
Base / Home Currency: GBP/USD = 1.584
Then,
Units = (100 * 20) / 1.584
Units = 1262
